# HoGentProjectenII-2024-DavidDemoen-EP3-backend

## General setup
- install project
```
yarn install
```
- run project
```
yarn start
```
## .env
Create a .env file with the following content:
```
NODE_ENV=development
DATABASE_USERNAME=""
DATABASE_PASSWORD=""
```
## Database Schema
